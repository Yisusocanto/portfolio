<div id="the-problem">
## THE PROBLEM
</div>

For video game enthusiasts, there was no minimalist, community-focused platform in the style of Letterboxd (for movies) or Goodreads (for books). Existing alternatives are usually ad-saturated news portals or digital storefronts (like Steam) where reviews get lost among memes and performance complaints. The challenge lay in designing a space dedicated purely to logging game activity, cataloging titles, and fostering honest review sharing.

<div id="the-solution">
## THE SOLUTION
</div>

Pixel Reviews solves this by providing a modern, clean, and content-centric interface. Users can search for video games using the IGDB database, mark them as played, add them to their wishlist, assign a rating, and write a review. A monthly log was implemented to view gaming history, along with a profile system showcasing user gaming stats (total games, average rating, favorite genre).

<div id="technical-challenges">
## TECHNICAL CHALLENGES
</div>

Several significant challenges were faced during the development of this Full-Stack application:

- **IGDB API Synchronization**: Consuming video game data in real time required configuring a cache layer in PostgreSQL and handling external API rate limits to prevent bottlenecks when loading the catalog.
- **Real-Time Average Calculation**: Computing average ratings from thousands of users and generating star distribution histograms required optimizing relational queries using SQL indexing and aggregation via SQLAlchemy in Flask.
- **Interactive User Experience**: To make logging a game instantaneous, a global state synchronization system with React Context and modal forms was implemented on the Frontend, minimizing unnecessary page reloads.
